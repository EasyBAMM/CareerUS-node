## :gear: 개발 환경 설정

1) Git clone project
2) Download node-modules
3) Start front, back-end 
4) Run these codes.
   
```sh
git clone https://github.com/EasyBAMM/CareerUS-node.git
```
```sh
cd front-end && npm install
cd back-end && npm install
```
```sh
cd front-end && npm start:dev
cd back-end && npm run start
```
